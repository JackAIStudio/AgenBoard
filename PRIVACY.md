# 隐私说明

AgenBoard 不包含广告、用户画像、跨 App 跟踪或第三方分析 SDK。项目本身不运营账号系统或数据收集服务。

## 数据如何处理

- 使用 Apple 本机识别时，录音和转写由系统语音能力处理；AgenBoard 不会把内容发送到项目维护者的服务器。
- 只有用户主动选择“阿里云 Fun-ASR”后，App 才会将本次录音上传到阿里云百炼（DashScope），并把当前启用的热词发送给该服务。数据处理受阿里云相应服务条款和隐私政策约束。
- 阿里云 API Key 保存在设备钥匙串。当前实现从 App 直接访问阿里云，适合个人自用和开发验证；面向公众分发前应改为由可信服务端管理凭证。
- 识别历史、录音、热词、快捷短语、设置和 Rime 用户词典默认保存在设备本地或 App Group 容器中。
- 键盘扩展自身不实现网络请求。开启“允许完全访问”是为了让键盘通过 App Group 与主 App 共享语音状态、识别结果、设置和快捷短语。

## 导入与导出

用户可以主动导出数据 ZIP。原始录音和 API Key 默认不会导出；只有用户分别明确开启对应选项时才会加入数据包。包含 API Key 的 `credentials.json` 是明文文件，应像密码一样妥善保管。

## 删除与控制

用户可以在 App 中删除识别历史、录音、热词、快捷短语和已保存的阿里云配置。卸载主 App 会移除其本地容器；系统对 App Group 和钥匙串数据的保留行为可能与普通容器不同，如需彻底清理请先在 App 内删除相关内容。

## Apple 隐私清单

主 App 和键盘扩展均包含 `PrivacyInfo.xcprivacy`。清单按当前实现声明：无跟踪；阿里云模式可能发送音频和用户内容用于 App 功能；同时声明项目实际使用的 UserDefaults、文件元数据和系统启动时间类 API 的用途。

隐私问题或疑似漏洞请按 [`SECURITY.md`](SECURITY.md) 中的私密渠道报告。
